# Fighting_game
Basic 2d fighting game with Javascript with reactangular sprites as players.



## How to Play
**Run index.js to play.**

### Player_1 (green) movements : 

A : move to left

D : move to right

S : jump

F : attack


### Player_2 (red) movements : 

arrow-left : move to left

arrow-right : move to right

arrow-up : jump

arrow-down : attack
